# Swasthya-Sakhi
‘Swasthya Sakhi’ Hub is a mobile-first platform connecting ASHAs/ANMs with SHGs to improve rural health. It enables real-time data reporting, AI-based alerts, gamified SHG engagement, and smart inventory. Officials get dashboards, heatmaps, and predictive insights for proactive care, ensuring faster response, better outreach, and no stockouts.
